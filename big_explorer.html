MindSentences2025 — linear decoding explorer

One interactive figure for the whole linguistic hierarchy (Acoustic → Phoneme → Syllable → Word), read from the cached SlidingWindow scores. Use the four menus below to slice the curves; they compose (logical AND).

  • Feature / Subscore / Type / Condition menus — filter which curves show.
  • Line style = test condition: Perception = solid, Production = dotted.
  • Opacity = decoding type: direct (train==test) opaque, transfer (train on the other condition) faded.
  • Colour = subscore: all (grey), content words (green), position-in-sentence 0 / 1 / >0 (purples).
  • Legend — still click to toggle / double-click to isolate within the current slice.
